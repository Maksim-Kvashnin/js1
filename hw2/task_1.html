1. Дан код:
var a = 1, b = 1, c, d;
1. c = ++a; alert(c);           // 2
2. d = b++; alert(d);           // 1
3. c = (2+ ++a); alert(c);      // 5
4. d = (2+ b++); alert(d);      // 4
5. alert(a);                    // 3
6. alert(b);                    // 3
Почему код даёт именно такие результаты?

ОТВЕТЫ: 
Если операция используется как постфикс, с оператором после операнда (например, x++), значение операнда возвращается, а затем увеличивается на единицу.
Если используется префиксная форма с оператором перед операндом (например, ++x), значение операнда возвращается увеличенным на единицу.

1. сначала увеличивается а до 2, затем присваивается в с

2. сначала b присваивается d, затем b увеличивается до 2

3. сначала увеличивается а на 1 до 3 (так как в 1. она уже увеличилась до 2), затем складывается с 2 и присваивается в с

4. сначала b (в 2. она уже увеличилась до 2) складывается с 2 и присваивается в d, затем b увеличивается на единицу.

5. результат всех операций префиксных и постфиксных в 1. и 3.

6. результат всех операций постфиксных в 2. и 4.
