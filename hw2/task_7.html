7. *Сравнить null и 0. Попробуйте объяснить результат.

ОТВЕТ: В JavaScript null это просто специальное значение, которое представляет собой «ничего», «пусто» или «значение неизвестно». Значения null и undefined равны друг другу, но не чему бы то ни было еще. Это жесткое правило прописано в спецификации языка. При явном преобразовании в число (то есть вызванном пользователем) null принимает значение 0
